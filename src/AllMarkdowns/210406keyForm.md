# 列表 & key
**创建时间：2021-04-06**

1. key帮助React识别哪些元素改变了，如被添加或删除。因此给数组中的每一个元素赋予一个确定的标识。

2. key最好是独一无二的的字符串，最好不要使用索引index作为key。

3. key只是在兄弟节点之间必须唯一，`在全局不需要唯一`。当我们生成两个不同数组是，可以使用相同的key值。

4. JSX允许在大括号中`嵌入任何表达式`
